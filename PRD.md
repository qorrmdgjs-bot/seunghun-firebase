# 사주풀이 웹사이트 PRD (Product Requirements Document)

## 1. 프로젝트 개요

### 1.1 비전
20년 경력 사주 명리 전문가의 관점을 담은 현대적 사주풀이 서비스. 과장이나 단정 없이 **통계적/경향적 관점**에서 해석하며, 미래를 **가능성의 흐름**으로 설명하는 신뢰할 수 있는 사주 분석 플랫폼.

### 1.2 핵심 철학
- **과장 금지**: "반드시 ~합니다" 대신 "~하는 경향이 있습니다"
- **단정 금지**: "~입니다" 대신 "~일 가능성이 높습니다"
- **통계적 접근**: 수천 년간 축적된 명리학 데이터 기반 경향성 분석
- **가능성 제시**: 미래는 고정된 것이 아닌 흐름으로 설명

---

## 2. 핵심 기능

### 2.1 사주 입력 시스템
- **생년월일시 입력**: 양력/음력 선택, 시간 모를 경우 처리
- **성별 선택**: 대운 계산에 필요
- **출생지 입력** (선택): 시차 보정용

### 2.2 사주팔자 분석 결과

#### A. 기본 사주 구성
- 연주, 월주, 일주, 시주 (4주 8자)
- 천간/지지 한자 및 한글 표기
- 오행 분포 시각화 (목/화/토/금/수)
- 음양 밸런스 표시

#### B. 3대 관점 분석

**1) 직업/재능 관점**
- 타고난 성향과 적성
- 어울리는 직업군 (통계적 경향)
- 주의해야 할 직업 환경
- 리더십 vs 전문가 성향

**2) 재물/금전 관점**
- 재물 운의 흐름 경향
- 재물을 모으는 스타일
- 투자 성향 분석
- 주의해야 할 금전 패턴

**3) 인간관계/대인 관점**
- 대인관계 스타일
- 어울리는 사람 유형
- 갈등 발생 가능 패턴
- 연애/결혼 궁합 경향

#### C. 시간의 흐름 분석

**대운 분석 (10년 단위)**
- 현재 대운 기간 표시
- 각 대운별 주요 테마
- 유리한 시기 / 주의할 시기

**세운 분석 (연도별)**
- 올해의 주요 흐름
- 월별 에너지 변화 (간략)

### 2.3 통합 비주얼 이미지 (핵심 차별화 기능)

**"나의 사주 여정" 한 장 이미지**
```
[과거] -----> [현재] -----> [미래 가능성들]
   |            |              |
 형성된       지금의          펼쳐질
 기반들       위치           흐름들
```

포함 요소:
- 오행 에너지 시각화 (색상 그라데이션)
- 현재 대운 위치 표시
- 미래 3-5년 키워드 배치
- 강점/약점 아이콘
- 저장/공유 가능한 PNG/JPG

## 3. 추가 제안 기능

### 3.1 오늘의 사주 팁
- 일진(日辰) 기반 간단 조언
- 매일 업데이트되는 맞춤 메시지
- 오늘 유리한 색상/방향/숫자

### 3.2 궁합 분석 (추후 확장)
- 두 사람의 사주 비교
- 관계 유형별 궁합 (연인/사업파트너/친구)
- 상호 보완점 및 주의점

### 3.3 질문형 상담 기능
- "올해 이직해도 될까요?" 같은 구체적 질문
- 해당 시기의 운세와 연결하여 조언
- 가능성의 흐름으로 답변

### 3.4 사주 용어 해설
- 어려운 한자/용어 팝업 설명
- 초보자도 이해 가능한 풀이

### 3.5 결과 저장 및 공유
- PDF 다운로드
- 이미지 저장
- SNS 공유 (요약 버전)

---

## 4. 기술 스택

### 4.1 프론트엔드
- **HTML5/CSS3/JavaScript** (바닐라)
- **Canvas API**: 통합 이미지 생성
- **반응형 디자인**: 모바일 우선
- **다크모드/라이트모드**: 시스템 설정 연동 및 수동 토글

### 4.2 사주 계산 로직
- **만세력 라이브러리**: 양력-음력 변환, 사주 계산
- **JavaScript 기반 명리 알고리즘 구현**

### 4.3 이미지 생성
- **Canvas API**: 커스텀 그래픽 생성

### 4.4 백엔드 (선택)
- Firebase (현재 연동 가능)
- 사용자 데이터 저장
- 결과 히스토리 관리

## 6. UI/UX 가이드라인

### 6.1 색상 테마
- 오행 기반 색상: 목(청), 화(적), 토(황), 금(백), 수(흑)
- 전체 톤: 고급스러운 한국적 분위기
- 라이트모드/다크모드 지원

### 6.2 폰트
- 제목: 서예체 느낌 (Noto Serif KR)
- 본문: 가독성 좋은 Noto Sans KR

### 6.3 인터랙션
- 부드러운 애니메이션
- 결과 순차 표시 (로딩 경험)
- 터치 친화적 UI

---

## 7. 구현 단계

### Phase 1: 기본 구조 (MVP) ✅
1. 입력 폼 UI 구현
2. 만세력 기반 사주 계산 로직
3. 기본 결과 표시 (4주 8자)
4. 오행 분포 시각화

### Phase 2: 분석 기능 ✅
5. 3대 관점 분석 텍스트 시스템
6. 대운/세운 계산 및 표시
7. 통합 비주얼 이미지 생성

### Phase 3: 부가 기능
8. PDF/이미지 다운로드 ✅
9. 오늘의 사주 팁 ✅
10. 다크모드/라이트모드 ✅
11. 사용자 저장 기능 (Firebase) - 추후

---

## 8. 검증 방법

1. **기능 테스트**: 다양한 생년월일로 사주 계산 정확도 확인
2. **UI 테스트**: 모바일/데스크톱 반응형 확인
3. **이미지 생성 테스트**: 다양한 브라우저에서 이미지 저장 확인
4. **사용자 피드백**: 결과의 명확성, 이해도 평가

## 10. 선택된 방향

- **디자인**: 전통+현대 믹스 (현대적 레이아웃에 전통 색상과 요소 조합)
- **언어**: 한국어 전용
- **회원 기능**: MVP는 비로그인, 결과는 로컬 저장/다운로드
- **이미지 스타일**: 인포그래픽 (아이콘, 차트, 키워드 배치)
- **테마**: 라이트모드 기본, 다크모드 지원 (시스템 설정 연동)
